## Capture tips (Unreal Editor)

1. Press **F9** for high-res screenshot (without UI), or use the viewport screenshot tool.
2. Hide debug overlays (` showflag.* `) before capturing.
3. Use **Alt+F10** (movie render queue) for staged shots if needed.
4. Export as PNG; keep width around 1920px for GitHub README embedding.
5. Compress large PNGs before commit (e.g. TinyPNG, ImageOptim) — stay under ~500 KB each when possible.
